第1课：Pandas基础与数据读取

20分钟 入门 已学习

学习目标

  • 理解Pandas的两种核心数据结构：Series和DataFrame
  • 掌握使用read_csv()读取CSV文件的各种参数
  • 掌握read_excel()读取Excel文件的方法
  • 了解Pandas支持的其他数据格式

知识点精讲

Series（系列）

一维标记数组，类似带索引的列表。每个元素可以是任意数据类型（int、float、string等）。

DataFrame（数据框）

二维标记数据结构，类似电子表格或SQL表。是数据分析中最常用的数据结构。

代码示例

import pandas as pd
import numpy as np

# 创建Series
s = pd.Series([1, 3, 5, 7, 9], index=['a', 'b', 'c', 'd', 'e'])
print("Series示例:")
print(s)

# 创建DataFrame
data = {
    'name': ['张三', '李四', '王五'],
    'age': [25, 30, 28],
    'score': [85, 92, 78]
}
df = pd.DataFrame(data)
print("\nDataFrame示例:")
print(df)

# 读取CSV文件
df = pd.read_csv('ecommerce_data.csv')
print(f"\n数据形状: {df.shape}")
print(f"列名: {df.columns.tolist()}")
print("\n前5行数据:")
print(df.head())

# 读取Excel文件
# df_excel = pd.read_excel('data.xlsx', sheet_name='Sheet1')

# 查看数据类型
print("\n数据类型:")

易错点

  • • read_csv()默认将第一行作为表头，使用header=0
  • • 文件路径要正确，Windows系统注意使用双反斜杠或正斜杠
  • • 大文件读取时可以使用chunksize分块读取
  • • 中文编码问题：使用encoding='utf-8'或'gbk'

业务解读

在电商场景中，我们通常从数据库或报表系统导出订单数据、客户数据到CSV/Excel文件。Pandas可以快速读取这些文件进行数据分析，这是数据分析师最常用的数据获取方式。

第3课：缺失值处理

30分钟 入门 未学习

学习目标

  • 理解缺失值的成因和对分析的影响
  • 掌握三种缺失值处理策略：删除、填充、插值
  • 学会根据业务场景选择合适的处理方法
  • 掌握fillna()和dropna()的常用参数

知识点精讲

策略一：删除法（dropna）

适用于缺失值较少且删除后不影响分析的情况。

# 删除含有缺失值的行
df_clean = df.dropna()

# 只删除特定列的缺失值
df_clean = df.dropna(subset=['order_id', 'customer_id'])

# 删除全部为空的行
df_clean = df.dropna(how='all')

# 删除缺失值比例超过50%的列
threshold = len(df) * 0.5
df_clean = df.dropna(thresh=threshold, axis=1)
策略二：填充法（fillna）

最常用的方法，可以用均值、中位数、众数或固定值填充。

# 用均值填充数值列
df['price'] = df['price'].fillna(df['price'].mean())

# 用中位数填充（对异常值更稳健）
df['quantity'] = df['quantity'].fillna(df['quantity'].median())

# 用众数填充类别列
df['category'] = df['category'].fillna(df['category'].mode()[0])

# 用0填充
df['discount'] = df['discount'].fillna(0)

# 用固定值填充
df['status'] = df['status'].fillna('未知')

# 用前一个值填充（前向填充）
df['date'] = df['date'].fillna(method='ffill')

# 用后一个值填充（后向填充）
df['date'] = df['date'].fillna(method='bfill')

# 用前后值的均值填充
df['price'] = df['price'].interpolate()
策略三：插值法（interpolate）

适用于时间序列数据，可以保持数据的连续性。

# 线性插值
df['price'] = df['price'].interpolate(method='linear')

# 时间加权插值（适用于时间序列）
df['price'] = df['price'].interpolate(method='time')

# 最近邻插值
df['price'] = df['price'].interpolate(method='nearest')

# 二次插值
df['price'] = df['price'].interpolate(method='quadratic')

易错点

  • • 盲目删除缺失值会导致大量数据丢失，应优先考虑填充
  • • 用均值填充可能拉低与异常值的差异，用中位数更稳健
  • • fillna()返回新的DataFrame，原数据不变，需要赋值
  • • 前向/后向填充可能传播错误，要检查填充结果

业务解读

在电商数据中，缺失值常见于：用户未填写的信息（地址、电话）、系统采集问题（价格、库存）、数据迁移丢失等。价格缺失时用均值填充是常见做法；订单状态缺失可能是系统延迟，可用前向填充。

df_unique = df.drop_duplicates(keep='last')

df_unique = df.drop_duplicates(subset=['order_id'], keep='first')

# 基于多列删除重复
df_unique = df.drop_duplicates(subset=['order_id', 'customer_id'], keep='first')

# 按某列排序后删除重复（保留最大值）
df_sorted = df.sort_values('date', ascending=False)
df_unique = df_sorted.drop_duplicates(subset=['customer_id'], keep='first')

import pandas as pd

df = pd.read_csv('ecommerce_data.csv')

1. 以下哪个方法可以查看DataFrame的基本信息？

正确答案：B

解析：df.info() 用于显示DataFrame的基本信息，包括数据类型、非空值数量、内存占用等。df.describe() 显示统计描述，df.head() 显示前几行，df.shape是属性不是方法。

2. 读取CSV文件时，中文乱码应该使用哪个参数？

正确答案：C

解析：encoding参数用于指定文件编码，UTF-8是通用的编码格式，GBK用于中文Windows系统创建的文件。

3. 删除含有缺失值的行，应该使用哪个方法？

正确答案：A

解析：dropna()用于删除含有缺失值的行/列，fillna()用于填充缺失值，isnull()用于检测缺失值，duplicated()用于检测重复值。

4. 用均值填充缺失值，适用于哪种情况？

正确答案：B

解析：均值只能用于数值类型数据的填充。类别数据用众数填充，日期数据用前向/后向填充更合适。

5. 以下哪个方法可以删除完全相同的重复行？

正确答案：D

解析：drop_duplicates()是Pandas专门用于删除重复行的方法，可以基于全部列或特定列去重。
