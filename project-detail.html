知识点精讲

Series（系列）

一维标记数组，类似带索引的列表。每个元素可以是任意数据类型。

import pandas as pd
s = pd.Series([1, 3, 5, 7], 
              index=['a','b','c','d'])
DataFrame（数据框）

二维标记数据结构，类似电子表格，是数据分析中最常用的数据结构。

df = pd.DataFrame({
    'name': ['张三','李四'],
    'age': [25, 30]
})

代码示例 

# 读取CSV文件
df = pd.read_csv('ecommerce_data.csv')

# 读取Excel文件
df_excel = pd.read_excel('data.xlsx', sheet_name='Sheet1')

# 查看数据基本信息
print(df.info())
print(f"\n数据形状: {df.shape}")
print("\n前5行数据:")
print(df.head())

# 查看数据类型
print("\n数据类型:")
print(df.dtypes)

易错点

  • • read_csv()默认将第一行作为表头
  • • Windows系统注意文件路径使用正斜杠
  • • 中文编码：使用encoding='utf-8'或'gbk'
  • • 大文件可使用chunksize分块读取

业务解读

在电商场景中，我们通常从数据库或报表系统导出订单数据、客户数据到CSV/Excel文件。Pandas可以快速读取这些文件进行数据分析。

数据探索与检查

25分钟 入门 已学习 ✓

学习目标

  • 掌握使用info()查看数据基本信息
  • 掌握使用describe()查看统计描述
  • 掌握isnull()检查缺失值
  • 掌握value_counts()统计分布

代码示例 

# 1. 查看数据基本信息（最常用！）
print("=" * 50)
print("1. 数据基本信息 (info)")
print("=" * 50)
df.info()

# 2. 查看数据统计描述
print("\n2. 统计描述:")
print(df.describe())

# 3. 查看前几行/后几行
print("\n前5行:")
print(df.head())
print("\n后3行:")
print(df.tail(3))

# 4. 检查缺失值
print("\n缺失值统计:")
print(df.isnull().sum())

# 5. 查看类别分布
print("\n类别分布:")
print(df['category'].value_counts())
print("\n类别占比:")
print(df['category'].value_counts(normalize=True).round(3) * 100, "%")

# 6. 查看列名和形状
print(f"\n数据形状: {df.shape}")
print(f"列名: {df.columns.tolist()}")

易错点

  • • describe()默认只显示数值列
  • • isnull()和isna()是等价的
  • • value_counts()默认降序排列
  • • head()和tail()默认显示5行

重点总结

  • • info()是最常用的数据概览方法
  • • describe()提供数值列统计摘要
  • • isnull().sum()快速定位缺失值
  • • value_counts()分析类别分布

缺失值处理

30分钟 入门 已学习 ✓

学习目标

  • 理解缺失值的成因和影响
  • 掌握三种缺失值处理策略
  • 学会根据业务场景选择方法
  • 掌握fillna()和dropna()参数

策略一：删除法（dropna） 

# 删除含有缺失值的行
df_clean = df.dropna()

# 只删除特定列的缺失值
df_clean = df.dropna(subset=['order_id', 'customer_id'])

# 删除全部为空的行
df_clean = df.dropna(how='all')

# 删除缺失值比例超过50%的列
threshold = len(df) * 0.5
df_clean = df.dropna(thresh=threshold, axis=1)

策略二：填充法（fillna） 

# 用均值填充数值列
df['price'] = df['price'].fillna(df['price'].mean())

# 用中位数填充（对异常值更稳健）
df['quantity'] = df['quantity'].fillna(df['quantity'].median())

# 用众数填充类别列
df['category'] = df['category'].fillna(df['category'].mode()[0])

# 用0填充
df['discount'] = df['discount'].fillna(0)

# 用固定值填充
df['status'] = df['status'].fillna('未知')

# 前向填充（前一个值填充）
df['date'] = df['date'].fillna(method='ffill')

# 后向填充（后一个值填充）
df['date'] = df['date'].fillna(method='bfill')

策略三：插值法（interpolate） 

# 线性插值
df['price'] = df['price'].interpolate(method='linear')

# 时间加权插值（适用于时间序列）
df['price'] = df['price'].interpolate(method='time')

# 最近邻插值
df['price'] = df['price'].interpolate(method='nearest')

# 二次插值
df['price'] = df['price'].interpolate(method='quadratic')

易错点

  • • 盲目删除会导致数据丢失
  • • 均值填充可能拉低差异
  • • fillna()需要赋值保存

业务解读

价格缺失时用均值填充是常见做法；订单状态缺失可能是系统延迟，可用前向填充。

选择建议

  • • 数值列：均值/中位数
  • • 类别列：众数/固定值
  • • 时间序列：插值/前向填充

  7. 保存清洗后的数据到 clean_data.csv

参考代码

import pandas as pd

# 1. 读取数据
df = pd.read_csv('ecommerce_data.csv')

# 2. 检查数据质量
print("原始数据形状:", df.shape)
print("缺失值:\n", df.isnull().sum())

# 3. 处理缺失值
df = df.dropna(subset=['order_id', 'customer_id'])
df['price'] = df['price'].fillna(df['price'].mean())
df['quantity'] = df['quantity'].fillna(1)

# 4. 删除重复
df = df.drop_duplicates(subset=['order_id'])

# 5. 处理异常值
df = df[df['price'] > 0]

# 6. 转换日期
df['date'] = pd.to_datetime(df['date'])

# 7. 添加总价列
df['total_price'] = df['price'] * df['quantity']

# 8. 保存
df.to_csv('clean_data.csv', index=False)
print("\n清洗后数据形状:", df.shape)

1以下哪个方法可以查看DataFrame的基本信息？

✓ 正确答案：B

解析：df.info() 用于显示DataFrame的基本信息，包括数据类型、非空值数量、内存占用等。df.describe() 显示统计描述，df.head() 显示前几行，df.shape是属性不是方法。

2读取CSV文件时，中文乱码应该使用哪个参数？

✓ 正确答案：C

解析：encoding参数用于指定文件编码，UTF-8是通用的编码格式，GBK用于中文Windows系统创建的文件。

3删除含有缺失值的行，应该使用哪个方法？

✓ 正确答案：A

解析：dropna()用于删除含有缺失值的行/列，fillna()用于填充缺失值，isnull()用于检测缺失值，duplicated()用于检测重复值。

4用均值填充缺失值，适用于哪种情况？

✓ 正确答案：B

解析：均值只能用于数值类型数据的填充。类别数据用众数填充，日期数据用前向/后向填充更合适。

5以下哪个方法可以删除完全相同的重复行？

✓ 正确答案：D

解析：drop_duplicates()是Pandas专门用于删除重复行的方法，可以基于全部列或特定列去重。

6以下哪些方法可以检测缺失值？（多选）

✓ 正确答案：A、B、C

解析：isnull()和isna()完全等价，用于检测缺失值返回True/False；notnull()是相反的检测；fillna()是填充方法，不是检测方法。

7fillna()方法可以使用哪些值进行填充？（多选）

✓ 正确答案：A、B、C、D

解析：fillna()非常灵活，可以填充固定值、统计值（均值、中位数、众数）、前后值（method='ffill'/'bfill'），也可以使用其他列或字典映射的值。

8关于drop_duplicates()方法，以下说法正确的有？（多选）

✓ 正确答案：A、B、C

解析：drop_duplicates()可以基于特定列（subset参数）去重，可以选择保留第一条（keep='first'）或最后一条（keep='last'），默认保留第一条。返回的是新DataFrame，不是原地修改。

9找出以下代码的错误并修正：

import pandas as pd
df = pd.read_csv('data.csv')
print(df.isnull)
df['price'].fillna(0)

错误及修正：

import pandas as pd
df = pd.read_csv('data.csv')
print(df.isnull())  # 错误1: isnull是方法，需要加括号
df['price'] = df['price'].fillna(0)  # 错误2: 需要赋值保存结果

解析：1) isnull是方法必须加括号调用；2) fillna返回新数据，需要赋值给原列才能保存修改。

10找出以下代码的错误并修正：

df = pd.read_csv('data.csv')
df.dropna()
print(df.shape)

错误及修正：

df = pd.read_csv('data.csv')
df = df.dropna()  # 需要赋值保存结果
print(df.shape)

解析：dropna()默认返回新DataFrame，原数据不变。必须赋值才能保存删除后的结果。

11找出以下代码的错误并修正：

df = pd.read_csv('data.csv', encoding='utf-8')
print(df['price'].mean())
df['category'] = df['category'].fillna(df['category'].mean())

错误及修正：

df = pd.read_csv('data.csv', encoding='utf-8')
print(df['price'].mean())
df['category'] = df['category'].fillna(df['category'].mode()[0])  # 类别列不能用mean()

解析：category是字符串类别列，不能使用mean()计算均值，应该使用众数mode()[0]进行填充。

12找出以下代码的错误并修正：

df = pd.read_csv('data.csv')
duplicates = df.duplicated()
df_clean = df.drop_duplicates()
print(f"发现{len(duplicates)}条重复记录")

错误及修正：

df = pd.read_csv('data.csv')
duplicates = df[df.duplicated()]  # duplicated()返回布尔Series，需要筛选
df_clean = df.drop_duplicates()
print(f"发现{len(duplicates)}条重复记录")

解析：duplicated()返回布尔Series（每行是否为重复），需要配合布尔索引筛选才能获取具体重复行。当前代码虽然不报错，但统计的是布尔值数量而不是重复行数量。
