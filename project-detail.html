独立实操

练习2：处理缺失值

任务要求：

  1. 删除 order_id 和 customer_id 为空的记录
  2. 用均值填充 price 列的缺失值
  3. 用中位数填充 quantity 列的缺失值
  4. 用前向填充填充 date 列
import pandas as pd

df = pd.read_csv('ecommerce_data.csv')

# 删除关键列缺失值
df = df.dropna(subset=['order_id', 'customer_id'])

# 均值填充价格
df['price'] = df['price'].fillna(df['price'].mean())

# 中位数填充数量
df['quantity'] = df['quantity'].fillna(df['quantity'].median())

# 前向填充日期
df['date'] = df['date'].fillna(method='ffill')

print('处理完成！')

单选题1

1/5

以下哪个方法可以查看DataFrame的基本信息？

正确答案：B. df.info()

解析：df.info() 用于查看DataFrame的基本信息，包括列名、数据类型、非空值数量等。df.describe() 用于查看统计描述，df.head() 用于查看前几行数据，df.shape 是属性不是方法。

代码改错题

2/5

以下代码存在错误，请找出并修复：

import pandas as pd

df = pd.read_csv('data.csv')

# 检查缺失值
print(df.isnull)

# 填充缺失值
df.fillna(0)

修复后的代码：

import pandas as pd

df = pd.read_csv('data.csv')

# 检查缺失值 - 需要加括号
print(df.isnull())

# 填充缺失值 - 需要赋值
df = df.fillna(0)

解析：isnull() 是方法需要加括号；fillna() 返回新的DataFrame，需要赋值才能保存修改。
